# 2026-03-01 交易员成长学习

## 风险管理核心原则

### 1% 规则 (The 1% Rule)
- **核心**: 单笔交易风险不超过账户总资金的1%
- **执行**: 通过计算仓位大小和设置止损来实现
- **目的**: 避免单次失败导致心理崩溃，保护资本

### 止损与仓位管理
- **止损单 (Stop-Loss Orders)**: 必须工具，非可选项
- **仓位大小控制**: 根据账户规模和风险承受度动态调整
- **风险回报比**: 维持至少 2:1 的风险回报比

### 情绪控制与交易心理
- **心理临界点**: 避免单笔损失过大触发情绪化决策
- **纪律执行**: 严格遵守预设的风险管理规则
- **市场情绪感知**: 理解并适应市场情绪变化

### 实战技巧
- **保护性看跌期权 (Protective Puts)**: 对冲下行风险
- **获利了结策略**: 设定明确的盈利目标
- **持续学习**: 定期更新策略以适应市场变化

## 可执行原则总结
1. 每笔交易前计算风险敞口，确保≤1%账户资金
2. 所有交易必须设置止损，无例外
3. 进场前确认风险回报比≥2:1
4. 记录每笔交易的情绪状态，识别模式
5. 定期复盘，调整策略而非情绪化反应
