 <section class="py-8 md:py-10">
 <h2 class="text-xl font-semibold mb-6 text-white">如何使用</h2>
 <div class="space-y-4">
 <div class="flex gap-4 bg-[#1a1a1f] border border-[#2d2d35] rounded-xl p-4 hover:border-primary">
 <span class="flex-shrink-0 w-7 h-7 bg-[#6cff32] text-[#0f0f12] font-bold text-sm rounded-full flex items-center justify-center">1</span>
 <div>
 <h4 class="font-medium mb-1 text-[#f5f5f7]">安裝 fourmeme CLI</h4>
 <p class="text-gray text-sm">在終端執行：<code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">pnpm add -g @four-meme/four-meme-ai@latest</code>。之後可用
 <code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">fourmeme <span class="before:content-['&lt;'] after:content-['&gt;']">command</span></code>
 呼叫。</p>
 </div>
 </div>
 <div class="flex gap-4 bg-[#1a1a1f] border border-[#2d2d35] rounded-xl p-4 hover:border-primary">
 <span class="flex-shrink-0 w-7 h-7 bg-[#6cff32] text-[#0f0f12] font-bold text-sm rounded-full flex items-center justify-center">2</span>
 <div>
 <h4 class="font-medium mb-1 text-[#f5f5f7]">安裝 four-meme-ai Skill</h4>
 <p class="text-gray text-sm">在終端執行：<code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">npx skills add four-meme-community/four-meme-ai</code>，將本
 Skill 安裝到你的 Agent / 環境中。</p>
 </div>
 </div>
 <div class="flex gap-4 bg-[#1a1a1f] border border-[#2d2d35] rounded-xl p-4 hover:border-primary">
 <span class="flex-shrink-0 w-7 h-7 bg-[#6cff32] text-[#0f0f12] font-bold text-sm rounded-full flex items-center justify-center">3</span>
 <div>
 <h4 class="font-medium mb-1 text-[#f5f5f7]">配置私鑰與 RPC（需簽名/發送交易時）</h4>
 <p class="text-gray text-sm">在<strong>你執行指令的專案目錄</strong>下放 <code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">.env</code>，寫入 <code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">PRIVATE_KEY=...</code>、可選 <code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">BSC_RPC_URL=...</code>。CLI
 會自動從當前目錄載入。如果你使用 OpenClaw，也可以在 Skill 管理頁中，將 four-meme-ai 的 apiKey 設置為你的 PRIVATE_KEY。</p>
 </div>
 </div>
 <div class="flex gap-4 bg-[#1a1a1f] border border-[#2d2d35] rounded-xl p-4 hover:border-primary">
 <span class="flex-shrink-0 w-7 h-7 bg-[#6cff32] text-[#0f0f12] font-bold text-sm rounded-full flex items-center justify-center">4</span>
 <div class="flex-1">
 <h4 class="font-medium mb-1 text-[#f5f5f7]">在對應 Agent 中啟用本 Skill / 插件</h4>
 <p class="text-gray text-sm mb-4 border-b border-dashed pb-4 border-[#2d2d35]">
 不同平台步驟略異，共通點：<strong>在對應平台中啟用 four-meme-ai</strong>，即可在對話中使用
 create、buy、sell、send、8004-register。</p>
 <div class=" space-y-3">
 <div class="border-b border-dashed pb-4 border-[#2d2d35]">
 <h4 class="text-[#8b5cf6] text-sm font-medium mb-1">OpenClaw</h4>
 <p class="text-gray text-sm">在 Skill 管理頁找到 four-meme-ai 並啟用，即可在對話中使用本 Skill。</p>
 </div>
 <div class="border-b border-dashed pb-4 border-[#2d2d35]">
 <h4 class="text-[#8b5cf6] text-sm font-medium mb-1">Cursor / Claude Code / 其他支援
 Skill 的編輯器</h4>
 <p class="text-gray text-sm">安裝本 repo 或 npm 套件後，在 Agent 設定中啟用 four-meme-integration
 skill，由 Agent 透過 fourmeme CLI 幫你呼叫相關指令。</p>
 </div>
 <div class="pb-4">
 <h4 class="text-[#8b5cf6] text-sm font-medium mb-1">Kimi、OpenAI 等（需可執行終端指令）</h4>
 <p class="text-gray text-sm">若該 Agent 能執行 Bash，可透過 <code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">fourmeme <span class="before:content-['&lt;'] after:content-['&gt;']">command</span></code>
 或 <code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">npx fourmeme <span class="before:content-['&lt;'] after:content-['&gt;']">command</span></code>
 呼叫。私鑰請僅透過安全方式提供，勿在對話中直接貼上。</p>
 </div>
 </div>
 </div>
 </div>
 <div class="flex gap-4 bg-[#1a1a1f] border border-[#2d2d35] rounded-xl p-4 hover:border-primary">
 <span class="flex-shrink-0 w-7 h-7 bg-[#6cff32] text-[#0f0f12] font-bold text-sm rounded-full flex items-center justify-center">5</span>
 <div>
 <h4 class="font-medium mb-1 text-[#f5f5f7]">在對話中直接下指令</h4>
 <p class="text-gray text-sm">例如：「幫我查 Four.meme 熱門榜」「報價買入 0.01 BNB 的某代幣」「創建一個名為 XXX 的 Meme
 代幣」等。Agent 會依 SKILL 說明呼叫 <code class="bg-[#222228] px-1.5 py-0.5 rounded text-xs">fourmeme config / token-rankings / quote-buy / buy / create-api / create-chain</code>
 等。</p>
 </div>
