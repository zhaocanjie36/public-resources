## Bug 修复 - API Key 显示和 OpenClaw 切换问题

**时间:** 09:16 - 09:31 EST

### 问题1: API Key 显示固定密码

**症状:**
- 前端从"已激活账户"列表复制 API Key 时，显示的是固定密码 `my-super-secret-password-Asd33520@!!`
- 实际上每个账号激活时都会生成随机的 API Key

**根本原因:**
- 前端 `ActiveAccounts.jsx` 使用硬编码的 `API_KEY` 常量
- 后端 `websocket.py` 和 `gateway.py` 返回 `config.PROXY_API_KEY` 而不是账号的真实 API Key

**修复内容:**

1. **前端修复** (`frontend/src/components/ActiveAccounts.jsx`):
   - 删除硬编码的 `const API_KEY = 'my-super-secret-password-Asd33520@!!'`
   - 修改复制按钮：`copyToClipboard(record.api_key, 'API Key')`

2. **后端修复** (`backend/api/websocket.py`):
   ```python
   # 修改前
   "api_key": config.PROXY_API_KEY
   
   # 修改后
   "api_key": account['api_key']
   ```

3. **后端修复** (`backend/api/gateway.py`):
   ```python
   # 修改前
   "api_key": config.PROXY_API_KEY
   
   # 修改后
   "api_key": account['api_key']
   ```

**验证:**
- 账号 jerseyaward0b 的真实 API Key: `0e590afe007220707c5614b7e8641ba7269a23726d1f5b3cbc80229b881abb5c`
- 前端复制按钮现在会复制真实的随机生成的 API Key

### 问题2: OpenClaw 切换失败

**症状:**
- 在 OpenClaw 配置卡片切换账号时报错
- 错误信息：`切换失败: [Errno 2] No such file or directory: 'openclaw'`
- 后续错误：`500: OpenClaw 重启失败: /usr/bin/env: 'node': No such file or directory`

**根本原因:**
1. 第一次尝试：直接调用 `openclaw` 命令，但 systemd 服务的 PATH 中没有该命令
2. 第二次尝试：使用完整路径 `/root/.nvm/versions/node/v24.14.0/bin/openclaw`，但该脚本依赖 `node`，而 `node` 也不在 PATH 中
3. 正确方案：使用 `systemctl --user restart openclaw-gateway` 直接重启服务

**修复内容:**

**文件:** `backend/api/openclaw.py`

修改两处重启命令（默认配置切换 + 账号切换）:

```python
# 修改前
result = subprocess.run(
    ['/root/.nvm/versions/node/v24.14.0/bin/openclaw', 'gateway', 'restart'],
    capture_output=True,
    text=True,
    timeout=30
)

# 修改后
env = os.environ.copy()
env['XDG_RUNTIME_DIR'] = '/run/user/0'
result = subprocess.run(
    ['systemctl', '--user', 'restart', 'openclaw-gateway'],
    capture_output=True,
    text=True,
    timeout=30,
    env=env
)
```

**关键点:**
- 使用 `systemctl --user` 需要设置 `XDG_RUNTIME_DIR=/run/user/0` 环境变量
- 直接重启 systemd 服务比调用 CLI 命令更可靠

```javascript
const [pageSize, setPageSize] = useState(10);

pagination={{
  current: currentPage,
  pageSize: pageSize,
  onChange: (page, size) => {
    setCurrentPage(page);
    setPageSize(size);
  },
}}
```
